# Lucas King's Portfolio Website

## Project Overview
Personal portfolio website for Chun-An (Lucas) King, showcasing career experience, education, projects, competitions, and certificates.

## Project Structure
- `index.html` - Main page (includes inline JS logic)
- `styles.css` - Dark theme stylesheet (Inter font, Indigo accent)
- `data.js` - All portfolio content data
- `images/` - Competition screenshots
- `certificates/` - Certificate images and PDF

## Deployment
- **GitHub Pages** - Static site hosting
- No build step required

## Tabs
1. About - Bio and social links
2. Experience - Work history
3. Education - Degrees and coursework
4. Projects - Accordion list with GitHub/report links
5. Competitions - Trading competitions with screenshots
6. Certificates - Finance and coding certifications

## Tech Stack
- Pure HTML/CSS/JavaScript (no framework)
- Google Fonts (Inter), Font Awesome 6
- GitHub Pages for hosting
